# 🧩 XML to SVG/PNG Converter

A simple Python script that converts Android Vector Drawable `.xml` icons into `.svg` and `.png` format. Useful for extracting high-quality icons from Android XML assets.

---

## 📁 Features

✅ Convert XML Vector Drawable → SVG  
✅ Convert SVG → PNG (512p resolution)  
✅ White-colored icons supported  
✅ Ideal for One UI / Material icons
